# 📊 Data Analysis and Visualization Project

## 📌 Project Overview
This project performs data cleaning, analysis, and visualization on a retail sales dataset to extract meaningful business insights.

## 🎯 Objectives
- Clean and preprocess raw dataset
- Perform exploratory data analysis (EDA)
- Identify sales and profit trends
- Visualize patterns using charts

## 📂 Dataset
Superstore Sales Dataset (public retail dataset)

## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Insights
- Technology category performs best in both sales and profit.
- Discounts negatively impact profitability.
- Sales vary significantly across regions.
- Strong seasonal trends observed in monthly sales.

## 📈 Visualizations Included
- Sales Distribution
- Sales by Category
- Profit by Category
- Sales by Region
- Monthly Sales Trend
- Discount vs Profit
- Correlation Heatmap

## 🚀 How to Run
1. Clone repository
2. Install requirements
3. Open Jupyter Notebook
4. Run cells sequentially
